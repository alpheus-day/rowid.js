# RowID CLI

CLI for RowID

## Installation

For NPM:

```bash
npx i -g rowid-cli
```

For Pnpm:

```bash
pnpm i -g rowid-cli
```

## Usage

Check the usage of RowID CLI with the following command:

```bash
rowid -h
```
